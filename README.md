# 🧪 Smart Lab Automation System

An automated Python-based pipeline for organizing, analyzing, visualizing, and reporting spectroscopy data (UV-Vis, PL, FTIR, lifetime, and drug sensing experiments).

This system transforms raw experimental data into:
- Structured datasets
- Scientific plots
- Trend analysis
- Journal-ready reports
- Interactive dashboard

---

# 🚀 Features

## 📂 Data Management
- Automatically organizes experimental files by:
  - Date (YYYY-MM-DD)
  - Experiment type (UV, PL, FTIR, lifetime)

## 📊 Spectroscopy Analysis
- UV-Vis absorbance analysis
- Photoluminescence peak detection
- FTIR spectrum handling
- Lifetime decay fitting

## 📈 Advanced Analytics
- Gaussian peak fitting
- Exponential decay fitting
- Calibration curve generation
- Limit of Detection (LOD) estimation

## 🧠 Trend Detection
- Peak shifts (red/blue shift)
- Intensity variation tracking
- Time-series comparison across experiments

## 📝 Reporting
- Automatic daily reports
- Weekly aggregated summaries
- Scientific interpretation of results
- Export to `.txt` (and optional `.docx` / PDF)

## 📊 Dashboard
- Streamlit-based interactive interface
- Visualize spectra and results
- Upload and analyze new data

## ⚙️ Automation
- Run-once pipeline execution
- Watch mode (real-time monitoring)
- Scheduled execution (daily automation)
- Windows Task Scheduler support
